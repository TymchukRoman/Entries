# Entries

<h4> 1. Download modules </h1>

<b> Make this in terminal </b>

<code> cd .\back\ </code>

<code> npm i </code>

<code> cd ../ </code>

<code> cd .\front\ </code>

<code> npm i </code>

<code> cd ../ </code>


<h4> 2. To run app, split you terminal and do the following.  </h1>

<b> In first: </b>

<code> cd .\back\ </code>

<code> npm run start </code>

<b> In second: </b>

<code> cd .\front\ </code>

<code> npm start </code>
